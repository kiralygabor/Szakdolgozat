Minijobz

(Kicsimelo)

A Modern Platform for Connecting Taskers and Employers

End-of-Year Software Project

Project Overview

Goal: Create a localized gig-economy marketplace.

Problem: Finding reliable help for small tasks is often difficult and unorganized.

Solution: A secure web platform connecting Employers with Taskers alongside real-time communication.

Code Spotlight: Task Creation


public function store(StoreTaskRequest $request) {
    // 1. Validation check
    $validated = $request->validated();

    // 2. Handle File Uploads (Secure Storage)
    $photos = [];
    if ($request->hasFile('photos')) {
        foreach ($request->file('photos') as $photo) {
            $photos[] = $photo->store('task-photos', 'public');
        }
    }

    // 3. Save Logic
    $advert = new Advertisment();
    $advert->fill($validated);
    $advert->photos = $photos;
    $advert->employer_id = Auth::id();
    $advert->save();

    return redirect()->route('my-tasks');
}

Demonstrates dependency injection, validation, and secure file handling.

Challenges & Solutions

Challenge: Real-time messaging without page reloads.

Solution: Implemented a Node.js sidecar server with Socket.io.


Challenge: Handling complex task states (Open -> Offers -> Accepted -> Completed).

Solution: State machine logic in Controllers + Database constraints.
